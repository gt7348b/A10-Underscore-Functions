Assignment A10 - Create jscript library

The john`.js file recreates several underscore js functions:

  • Filter: roscoe.filter(array, condition as function) - returns values matching condition
  • Reject: roscoe.reduce(array, condition as function) - returns values not matching condition
  • Initial: roscoe.initial(array) - returns first value of array
  • Last: roscoe.last(array) - returns last value of an array
  • Map: roscoe.map(array, function action) - returns an array with the action added to each item
  • Reduce: roscoe.reduce(array, [initial value]) - returns the sum of an array. An initial value is optional

Please open your console and you can access these functions with roscoe.'function you want' (i.e. roscoe.filter). A test array of var = test_array = [1, 2, 3, 4, 5, 6] has been created to test the functions
